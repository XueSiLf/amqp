# lavamusic-amqp

## 介绍

支持在 EasySwoole 框架中使用 支持 AMQP 协议的消息队列，如：RabbitMQ

支持消息发布和消费

## Composer

本项目可以使用composer安装，遵循psr-4自动加载规则，在你的 `composer.json` 中加入下面的内容:

```json
{
    "require": {
        "lavamusic/amqp": "1.0.0-alpha"
    }
}
```

然后执行 `composer update` 安装。

或者使用如下命令安装：
```bash
composer require lavamusic/amqp "1.0.0-alpha"
```
